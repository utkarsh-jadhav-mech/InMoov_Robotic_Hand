### InMoov Robotic Hand Project - README

This project involves the design and development of a 3D-printed robotic hand inspired by the InMoov platform. It uses servo motors, Arduino control, and a tendon-driven mechanism to achieve human-like finger movement.

## 🎯 Project Overview

- Design a functional robotic hand using 3D CAD tools
- Achieve human-like finger motion using servo actuation
- Implement Arduino-based control system
- Understand mechatronics integration (mechanical + electrical + software)
- Build a fully working prototype system
